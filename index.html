Célula BIM UFPE

Uma iniciativa para capacitar os alunos nos primeiros passos para o mercado de trabalho AECO inserindo o uso do BIM na universidade.

O projeto

A UFPE

Foi a segunda universidade brasileira selecionada pelo Ministério da Economia para implementar uma Célula BIM. Seu objetivo é criar instrumentos pedagógicos para a difusão do BIM no ambiente acadêmico. Projeto coordenado pelos pesquisadores do Laboratório de Estudos Avançados em Arquitetura - lA2: Prof. Max Andrade, Prof. Cristiana Griz, Prof. Luiz Amorim e Prof. Leticia Mendes.
